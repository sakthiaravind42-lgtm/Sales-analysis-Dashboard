**Pizza Sales Analysis Dashboard | SQL & Power BI
Project Overview**

This project involves the development of a comprehensive Pizza Sales Analysis Dashboard using SQL for data cleaning and preparation, and Microsoft Power BI for data modeling and visualization. The primary objective of the project is to analyze sales performance, customer purchasing behavior, and product-level trends to support data-driven business decisions.

The dashboard provides both high-level KPIs and detailed insights into sales trends, best- and worst-performing products, and peak ordering periods.

**Tools and Technologies**

SQL – Data cleaning, transformation, joins, and aggregations

Microsoft Power BI – Data modeling, DAX calculations, and dashboard development

DAX – KPI metrics and time-based analysis

**Data Cleaning and Preparation (SQL)**

Cleaned raw pizza sales data using SQL queries

Removed inconsistencies and handled missing values

Standardized date, numeric, and categorical fields

Created aggregated datasets using JOIN, GROUP BY, and WHERE clauses

Prepared optimized tables for Power BI reporting
